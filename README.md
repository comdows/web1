# YOU&I 목재 안내판 디자인 시스템

목재 안내판에 **실제 제작 가능한** 디자인을 만들기 위한 웹 도구입니다.
일반 이미지 생성 AI(GPT·Gemini 등)는 예쁜 결과는 주지만 우리 안내판
규격·재료에 맞는 결과를 보장하지 못합니다. 이 프로젝트는 그 간극을
메우는 것을 목표로 합니다.

## 지금 들어있는 것

| 파일 | 설명 |
|------|------|
| `editor.html` | 피그마식 웹 편집기 — AI/일러스트로 만든 안내판 이미지를 불러와 **수정·요소 추가·내보내기** |
| `editor.js` | 편집기 로직 (외부 의존성 없는 vanilla JS 캔버스) |
| `editor.css` | 편집기 UI 스타일 |
| `design-system.js` | **공통 규칙 시드** — 색/폰트/보드 규격/레이아웃 존/픽토그램을 코드화 |
| `README.md` | 이 문서 |

## 편집기 사용법

1. `editor.html`을 브라우저에서 엽니다. (설치·서버 불필요)
2. 상단에서 **보드 규격**(가로/세로 mm, 여백, 모서리, 바탕색)을 고릅니다.
3. 좌측에서 **텍스트 / 사각형 / 원 / 이미지 / 픽토그램**을 추가합니다.
4. 캔버스에서 **드래그로 이동**, **핸들로 크기조정**, **더블클릭으로 텍스트 편집**.
5. **PNG 내보내기**(인쇄용 300 DPI 가능) 또는 **저장**(JSON, 나중에 이어서 작업).

- 이미지/SVG는 캔버스로 **드래그&드롭**해도 들어갑니다.
- `Space+드래그` 화면 이동, `Ctrl+휠` 확대/축소, `Delete` 삭제.

### ⚠️ 일러스트(.ai) 파일 안내
브라우저는 `.ai`를 직접 못 엽니다. 일러스트에서 **SVG로 내보내면**
글자·도형·픽토그램이 벡터로 살아있어 편집기에서 그대로 불러올 수 있습니다.
(현재 SVG는 이미지 레이어로 배치됩니다. 벡터 단위 편집은 다음 단계 과제.)

## design-system.js — 공통 규칙의 단일 출처

현재 값은 참고 이미지(`design.jpg`)에서 뽑은 **임시 시드**입니다.
실제 일러스트 원본을 SVG로 올려 검증하면서 아래를 채워가면,
편집기 프리셋과 (이후) AI 프롬프트가 모두 이 한 파일을 따릅니다.

- `palette` — 목재/패널/포인트 색
- `fonts` — 제목/본문/라벨 폰트
- `boards` — 안내판 규격 프리셋 (mm)
- `layout` — 제목/본문/하단 존, 안전 여백, 최소 글자 크기
- `pictograms` — 화살표·화장실·주차 등 공통 아이콘 (SVG path)
- `components` — QR·플레이트 규칙

## 전체 로드맵 (단계적)

- [x] **1단계 — 웹 편집기** : 생성된 이미지 수정·요소 추가·내보내기 ← 현재
- [ ] **2단계 — 공통 규칙 확정** : 일러스트 원본(SVG)으로 `design-system.js` 검증·확장
- [ ] **3단계 — 데이터 적용** : 제목/항목/지도 데이터를 레이아웃에 자동 배치
- [ ] **4단계 — 외부 안내판 도입** : 타사 이미지를 우리 규격/스타일로 변환
- [ ] **5단계 — CAD/PDF 지도 변환** : 도면·PDF 지도를 안내판용 이미지로 가공
- [ ] **6단계 — AI 생성 가이드** : '제작 가능한' 안내판을 뽑는 프롬프트 규칙 정리

## 다음에 필요한 자료

`design-system.js`를 실제 값으로 채우려면:
- 일러스트 원본 → **SVG 내보내기** 몇 개 (용량 큰 .ai 대신)
- 실제 제작 치수(mm), 목재 종류·두께, 사용 폰트, 브랜드 색상
