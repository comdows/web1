## 1. 단기 — 로드맵 v2 계승 (Phase 1~6)

순서: **Phase 1 → 3 → 4 연속**, Phase 2·5·6은 외부 트리거 도달 시 인터럽트/병행. 각 페이즈 완료마다 결과 보고 후 다음 착수.

### Phase 1 — 성장 계측 기반 [완료 · 운영 중]
- `0034_growth.sql`(events.entity_type/id + metrics_weekly 스냅샷 + admin_snapshot_weekly RPC + 성장/코호트/머니패스 뷰) + metrics.mjs·metrics.yml 주간 cron + admin 성장 패널(시계열 CSS 바·코호트 그리드·머니패스, 차트 라이브러리 금지, "세션 기준 근사" 명기).
- 0034·0035·0036 운영 반영 완료. 최근 완결 주(2026-07-13 시작) 스냅샷은 세션 1·WAU 1·검색 0·외부방문 4이며 주간 누적을 계속한다.

### Phase 2 — 도메인 전환 [도구 완료 · 트리거: 도메인 구매]
- `scripts/switch-domain.mjs` 1회 + `domain-setup.md` 절차(DNS→Pages HTTPS→Supabase Auth redirect→GSC 신규 속성·주소 변경) → 301·canonical·OAuth 확인. 전환 주를 계측에 라벨링(색인 출렁임 2~6주를 성장 실패로 오판 방지).
- 도구와 체크리스트만 완료됐으며 실제 도메인 구매·DNS·서비스 주소 전환은 아직 실행하지 않았다.

### Phase 3 — EN 확충 604→1,719 [완료]
- EN 엔트리 1,719/1,719 + `en-coverage.mjs` 빌드 검증 + 분야 허브 인트로 45/45 완료. EN 뉴스 미러는 컷 → 정적 "Recently added"로 대체.

### Phase 4 — KO 편집 콘텐츠 + 재방문 루프 [시스템 완료 · 콘텐츠 지속]
- `articles.ko.json` → `/guide/<slug>/` 정적 페이지 + KO llms.txt + 홈 개인화 가이드 노출까지 완료. 편집 가이드 7개 분야와 도움말 8편이 있으며, 남은 38개 분야는 4주 이상 계측 후 월 2~4편 상한으로 선정한다.

### Phase 5 — 유동성 콜드스타트 [트리거: WAU 4주 연속 상승 · Phase 1 패널로 판정]
- 공급 100% 데모 탈출: 등록 퍼널 마찰 제거(머니패스 이탈 실측 후) · 운영자 클레임 승인자 등록 크로스 유도 · outreach 운영 플레이북 문서화 · 데모 전환 기준 문서화. 수집기→제휴 자동 생성은 **부결**(중립·익명 충돌).
- 최근 확정치가 WAU 1·검색 0이고 4주 연속 상승 데이터가 없어 미착수 상태를 유지한다.

### Phase 6 — 수익화 준비 트랙 [시스템 준비 완료 · 외부 조건 대기]
- 공지 배너·서버/프론트 이중 스위치·legal 실명 교체 절차·무통장 리허설·TERMS_VERSION 개정 절차를 구현·문서화했다. 통신판매업 신고와 30일 사전 공지는 미실행이며 스위치는 계속 off다.

## 2. 장기 (12~24개월) — 신설 방향

로드맵 v2 너머, 이번 사업·제품 조사에서 도출한 후보. 각 후보에 [트리거]·[선행조건]·[리스크]·불변원칙 부합을 명시.

### L1. 이메일 레이어 2종 활성화 [코드·DB 완료 · Phase 2 종속]
알림 리마인더(0024, 하루 1통 상한·수신거부 구현됨) + 제휴 서버 발송(0015, 이중 게이트·일 10건 상한). **선행**: 발신 도메인 SPF/DKIM/DMARC 인증(github.io로는 불가 → 도메인 구매 필수) · Resend 키 · 처리방침·약관 개정. **도메인 전환(Phase 2)과 한 묶음 진행 권장.** 불변원칙: 정보통신망법 §50 수신거부 준수.

### L2. 비컨헤드(크라우드펀딩) 깊이 증명 [부분 구현 · 계측 트리거]
가이드와 유형별 분류 설명은 구현됐다. 리워드/증권/대출/기부 전용 판단필드와 전용 비교표는 부분 미구현이며 Phase 1에서 크라우드펀딩 검색 수요가 확인될 때 확장한다. **리스크**: 증권형·대출형은 근거법·모금한도가 다른 별개 제도 → **정보 안내만, 접수·소개·자문 없음**(투자중개 경계).

### L3. 운영자 셀프서브 심화 [부분 구현 · Phase 5 종속]
운영자 대시보드와 플랫폼 Q&A 답변은 구현됐다. 소식 셀프 등록(검수 유지)·Q&A 답변 통계·제안 응답률 지표가 남았다. **선행**: 운영자 인증 수 증가(Phase 5 유동성). **리스크**: 소개문·정보 직접 수정은 계속 "정정 제안 → 검수" 경유(데이터 품질 방어선 — 의도된 설계 유지).

### L4. 자동등재 점진 개방 [활성 · 유지 확정]
`autolist`(0016)는 `enabled:true`, `min_confidence:80`으로 유지한다. 수집 예산과 검수 큐를 국내/해외×main/ad 네 버킷으로 완전히 분리하고, ad는 점수와 관계없이 수동검수 전용이다. main만 directUrl·분야·임계값 검증 후 `lifecycle=review`로 등재하며 0043이 서버에서도 pool·국가 일치를 재검증한다. 2026-07-21 실행에서 23건이 자동등재됐고 현재 사후 검수 대기 44건이다. **리스크**: 디렉토리 중립·품질 — 확정 전 URL·분야·소개문 확인을 생략하지 않고 검수 적체를 주간 운영 루틴에서 해소한다.

### L5. 계측 기반 버티컬 재우선순위 [확정 · 가정 검증 장치]
Phase 1 머니패스 데이터로 화폐화용 첫 버티컬(현재 가정: 유통·프리랜서)을 실측 재조정. 사업계획 §3의 "비컨헤드↔화폐화 버티컬 분리" 가정을 검증·수정하는 상시 루프.

### L6. 특허 4건 출원 [확정 · 시한 2027-07]
발명 설명서 초안만 존재하며 출원은 미완료다. 2026-07-03 공개 기준 공지예외 12개월 내 변리사 선행조사·출원. **지연 시 권리 상실.** 우선순위: 발명 1(익명성 자동검증, 구현 완료)·발명 4(밴드 가치진단, 법적 조건).

### L7. 수익화 스위치 온 [기능 준비 · 운영/법적 게이트 off]
게이트(G1~G3·X1·X2) + 통신판매업 신고 + 컴플라이언스 체크리스트 완료 후에만. 기능은 준비됐지만 서버+프론트 이중 게이트를 모두 off로 유지한다. **개발 범위가 아니라 운영 판단** — 30일 전 공지 후 시작.
