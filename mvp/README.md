# 플랫폼올 (가칭) — 분야별 플랫폼 디렉토리 (MVP)

사업자·개인이 "나가서 붙을 수 있는 플랫폼"을 **분야별로 정리한 디렉토리**입니다.
수수료·정산 상세 비교가 아니라, **어떤 분야에 어떤 플랫폼이 있는지 + 개략 설명**만 빠르게 훑는 것이 목적입니다.

## 실행
```
# (선택) 분야별 정적 페이지·sitemap 생성
cd mvp && node build.js

# 로컬 서버
python3 -m http.server 8000      # → http://localhost:8000
#  Windows: python -m http.server 8000  /  또는 그냥 index.html 더블클릭
```

## 구성
| 파일 | 역할 |
|---|---|
| `index.html` | 디렉토리 홈 — 검색 + 분야 필터칩 + 분야별 카드(이름·개략설명·지역·공식링크) |
| `data/platforms.js` | 분야(12) + 플랫폼(60) 데이터. 각 플랫폼은 `blurb`(한 줄 소개) |
| `build.js` | 분야별 정적 페이지 `c/{분야}.html`(색인용, JSON-LD ItemList) + sitemap 생성 |
| `assets/style.css`, `assets/app.js` | 스타일 · 공용 로직 |

> 소규모·신생 플랫폼까지 웹 검색으로 폭넓게 수록했습니다. URL·설명은 검색 기반이라 일부 변동
> 가능성이 있으니, 이상하면 알려주시면 수정합니다.

## 원칙
- 각 설명은 **개략 소개(중립·사실)**. 수수료·정산·입점 조건 등 상세는 각 플랫폼 공식 사이트에서 확인.
- 플랫폼 추가·수정은 `data/platforms.js`만 편집하고 `node build.js` 재실행.

> 참고: 이전 버전의 상세 비교·계산기 기능은 방향 전환(디렉토리 중심)에 따라 제거했습니다.
> 필요 시 git 이력에서 복구 가능합니다.
