# 세모플 3단계 (플랫폼 거래소) 구체화 기획 v1

> 전제: 세모플 거래소는 **"익명 매물 정보의 게시 + 쌍방 동의 시 소개"까지만** 한다.
> 중개·자문·실사·가치평가는 하지 않는다. 이 문서는 그 이유(법적 근거)와 실제 작동 구조를 정의한다.

---

## 1. 왜 "중개"를 하면 안 되는가 (핵심 논리)

### ① 무인가 투자중개업 — 형사처벌 영역
플랫폼 매각의 대부분은 법적으로 **주식(지분) 양수도**다. 주식은 자본시장법상 **증권**이고,
타인 간 증권 매매를 **"중개·주선·대리"** 하는 행위는 **투자중개업**으로 금융위원회 **인가**가 필요하다.
인가 없이 영업하면 **무인가 금융투자업 — 형사처벌 대상**(자본시장법상 5년 이하 징역 수준의 중형)이다.
- 세모플이 매도·매수인을 붙이고 **가격·조건 협상에 개입**하거나 **거래 성사를 주선**하면 이 경계를 넘을 수 있다.
- 경영권 이전을 수반하는 M&A 중개는 규제가 완화된 영역이 있다는 업계 논의도 있으나, 소규모 플랫폼 거래는
  "비상장 주식 양수도"와 경계가 모호하다 → **보수적으로 선을 긋는 것이 유일하게 안전**하다.

### ② "중개"인지 아닌지는 표현 + 보수 + 개입 정도로 종합 판단된다
규제기관·법원은 간판이 아니라 실질을 본다. 세 가지가 함께 가야 안전하다:
| 판단 요소 | 위험 신호 | 세모플의 선 |
|---|---|---|
| **표현** | "M&A 중개·자문·매칭 성사" | "익명 정보 게시·연결" 로만 표기 |
| **보수** | 거래액 연동 성공보수(%) — 사실상 중개 보수로 재해석 | **정액 리스팅료·열람료만**, 성사·거래액 절대 비연동 |
| **개입** | 가격 제시·협상 참여·계약 관여·실사 수행 | 쌍방 의사 확인 후 **소개(연락 연결)에서 종료** |

### ③ 자문·실사·가치평가는 각각 별도의 전문자격 영역
- 투자 판단 조언 = **투자자문업**(등록 필요) / 기업가치 평가·실사 = 회계·감정 전문 영역.
- 무자격 수행은 그 자체로 리스크이며, 잘못된 평가로 손해가 나면 **배상 책임**이 세모플로 온다.

### ④ 책임의 전이 — 중개자로 인식되는 순간 분쟁의 당사자가 된다
정보 게시판+연결자 지위면 책임 범위가 "게시 관리·소개 절차"로 한정된다.
중개자로 인식되면 매물 정보 허위, 가치 산정 오류, 거래 실패·하자까지 **민사 분쟁의 피고**가 될 수 있다.
(약관 면책도 고의·중과실은 못 막는다 — 약관규제법 §7.)

### ⑤ 회원 DB·개인정보 — 우리가 만지면 안 되는 자산
플랫폼 매각에는 회원 DB 이전이 따라오는데, 이는 **개인정보보호법 §27(영업양도 시 통지 의무)** 절차가 필요한
당사자 간 법률 행위다. 세모플이 DB 이전에 관여하면 개인정보 처리 책임이 발생한다 → **절대 비관여**.

> **한 줄 결론**: "중개를 안 하는 것"은 소극적 회피가 아니라 **사업 모델 그 자체**다.
> 정보 게시·연결에 머무는 대가로 세모플은 인가 없이, 분쟁 책임 없이, 정액 이용료 수익으로 거래소를 운영할 수 있다.
> 이 선은 이용자에게도 이익이다 — 계약·실사는 자격 있는 전문가(로펌·회계법인)가 맡는 것이 안전한 거래다.

## 2. 서비스 구조 구체화

### 2-1. 공급(매각) 측 — 구조화된 익명 리스팅
| 필드 | 예 | 규칙 |
|---|---|---|
| 코드명 | D-104 | 세모플이 부여 |
| 분야 | 콘텐츠·창작 수익화 | 35개 분야 체계 사용 |
| 지역 | 국내 | 광역 단위만 |
| 연매출 밴드 | 1~5억 | 정확 수치 금지, 밴드만 |
| 희망 형태 | 지분 전량 / 지분 일부+운영승계 / 자산 양수도 | 선택식 |
| 익명 요약 | "운영 6년차 수공예 버티컬…" | 익명성 규칙 통과 필수 |
| 하이라이트 | "작가 풀 보유 · 재방문율 상위" | 범주·밴드 표현만 |
| 매각 사유(선택) | 운영자 이직 | 신뢰 재료 |
| **희망 가격** | — | **게시하지 않는다.** 가격은 소개 후 당사자 협상(가격 제시·조정에 세모플 개입 금지) |

**익명성 규칙(검수 기준)** — 게시 전 전부 통과:
- [ ] 플랫폼명·URL·상호·대표자를 유추할 수 있는 표현 없음
- [ ] 구체 수치(매출·회원 수) 대신 밴드/범주
- [ ] 지역은 광역 단위, "OO시 유일" 류 식별 표현 금지
- [ ] 연락처·개인정보 없음 / 금지 업종 아님
- [ ] 검증 가능성: 등록자가 실제 운영자임을 비공개 채널로 확인(도메인 이메일 등)

### 2-2. 수요(인수) 측 — 인수 희망 브리프(Buyer Brief)
인수·투자 희망자가 **관심 분야 · 예산 밴드 · 희망 형태 · 주체(개인/법인)** 를 등록
→ 조건에 맞는 신규 매물 등록 시 우선 알림(오픈 후). 매물이 얇은 초기에 **수요 풀을 먼저 쌓는 장치**.

### 2-3. 프로세스 (7단계 SOP)
1. **접수**(비공개 채널) → 2. **검수·익명화**(위 규칙, 세모플이 코드명 부여·요약 재작성) → 3. **게시**
→ 4. **관심 수집**(인수자 관심 등록 + 브리프 매칭 알림) → 5. **양측 의사 확인**(각각 비공개로)
→ 6. **NDA 권고 후 소개** — 상호 동의 시에만 연락 연결. 표준 NDA 양식 안내(참고용, 법률자문 아님 고지)
→ 7. **당사자 + 전문 자문사 직접 진행**(협상·실사·계약·정산). 성사 자진신고 시 리스팅 종료 + 익명 통계 반영

**세모플의 역할은 6번에서 끝난다.** 7번에 관여하는 순간 §1의 선을 넘는다.

### 2-4. 정보 콘텐츠 (자문 아님 — 일반 정보 고지 필수)
- **거래 형태 가이드**: 지분 양수도(회사 통째, 부채 포함) vs 자산·사업 양수도(자산 선별 인수, 회원DB 이전 시 개인정보보호법 §27 통지 필요) 차이
- **매각 준비 체크리스트**(기존) + **인수 실사 항목 개요**(매출 증빙·트래픽·계약·분쟁 여부 — "자문사와 확인하세요")
- 전문 자문사 안내는 **디렉토리 소개 수준**(법률·세무 분야 플랫폼 링크)까지만

## 5. v2 확정 — 자산·사업 양수도 한정 + 지분 분기 (2026-07)

§1의 리스크를 구조적으로 제거하기 위해 **거래소의 직접 취급 범위를 자산·사업 양수도로 한정**했다.
지분(주식) 매매·투자유치·주식 거래는 게시·소개 자체를 하지 않고 전문 자문(로펌·회계법인) 이용 안내로만
분기하며, 안내 대가(리퍼럴 피)는 받지 않는다 — 특정 자문사 소개·매칭도 하지 않는다("소개의 실질" 회피).

**지분 3중 차단(전부 클라이언트+운영, 서버 스키마 불변):**
1. 입구 — 매각 접수·브리프 폼의 "거래 트랙" 선택에서 지분 선택 시 접수 대신 자문 안내 카드 렌더
2. 중간 — 보드에서 지분 mode 매물(구조 확정 전 게시분)은 관심 등록 차단 + 안내문, 데모 데이터 자산화
3. 출구 — 관리 콘솔 매물 검수에서 지분 접수 건은 "게시 불가" 배지 + 게시 버튼 잠금 + 반려 사유 프리셋

**자산 양수도 트랙 심화(에이전트 워크스루 21개 막힘 반영):**
- 접수 폼: 이전 자산 체크리스트(도메인·코드·상표·회원DB[§27 고지]·계약·SNS·재고), 운영 인수인계(없음/1/3개월),
  매각 사유 필수 프리셋+보충, 비공개 검증 자료(게시·공유 금지 — 운영자 확인 후 "운영자 확인 ✓" 칩),
  매도자 이메일 공유 동의 필수(§2 제3자 제공 정합) — 전부 `deal_submissions.payload`(jsonb)로 SQL 0건
- 검수: 자산·인수인계가 하이라이트 칩 초안에 자동 합류 → 게시 카드에 노출
- 소개 SOP 2단: ① 매도자 의사 확인 메일(관심 익명 요지, 단독 수신) → ② 소개 초안(양측, NDA 권고·§27·
  양수도 가이드 링크 포함). 소개 완료 버튼은 초안 발송 후에만 활성화
- `?view=deal-guide` 양수도 가이드 페이지: 소개 후 5단계(NDA→자료 교환→실사·계약→§27→대금·이전) +
  참고용 NDA 조항 체크리스트(법률자문 아님 고지)
- 브리프 "우선 안내"의 인앱 이행: 내 활동에서 활성 브리프 × 공개 매물 조건 대조 배너, 관리 콘솔 브리프에 맞는 매물 코드 표시
- 검수 SLA 고지(보통 3영업일) + 내 활동 경과일 표시 + 5일 경과 시 문의 mailto
- 약관 §1에 자산 양수도 한정·지분 미취급 명문화, TERMS_VERSION 2026-07.1
